About

The Excel templates are automatically regenerated based on fields defined in the NANoREG templates. The generation process includes cleaning of the JRC fields and facilitates the automatic generation of JSON configuration files, necessary to enable import into an eNanoMapper database instance

While not strictly following the ISA-TAB and ISA-TAB-Nano formats, the templates have been designed around ISA-Tab-logic, i.e. structuring the data in investigation-study-assay related groups.
