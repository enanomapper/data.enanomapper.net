Implementation

The eNanoMapper prototype database is part of the computational infrastructure for toxicological data management of engineered nanomaterials, developed within the EU FP7 eNanoMapper project.

  • User friendly data entry templates

  • Provides support for upload, search and retrieval of nanomaterials and experimental data through a REST web services API and a web browser interface.

  • Implemented by a customized version of AMBIT web services.
